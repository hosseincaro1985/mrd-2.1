# هدف پروژه

این پروژه یک بازی آموزشی فارسی برای دانش‌آموزان دبستان است.

هدف بازی آموزش مدیریت احساسات از طریق داستان، گفتگو، سوال و تعامل با شخصیت آدم فضایی است.

این پروژه در حال توسعه است.

تمام تغییرات باید فقط مطابق این فایل انجام شوند.

---

# قانون شماره 1

هیچ مدل هوش مصنوعی اجازه ندارد بدون اجازه کاربر:

- ساختار پروژه را تغییر دهد.
- فایل جدید ایجاد کند.
- فایل حذف کند.
- نام فایل‌ها را تغییر دهد.
- معماری پروژه را تغییر دهد.

ابتدا باید گزارش بدهد.

سپس منتظر تایید کاربر بماند.
